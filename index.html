### Features Highlight 1. **Home Page**: Displays news articles with an Unsplash image (`fetchUnsplashImage()`). 2. **Admin Dashboard**: Modal to manage items (add/edit/delete). 3. **Persistency**: News data persists using `localStorage`. 4. **Modern UI**: Tailwind CSS is used for styling. 5. **Burmese Language**: UI labels and placeholders are localized into Burmese.
